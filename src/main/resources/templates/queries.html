Post queries

Post queries are a string of ANDed together, space-separated conditions of one or more of the following types:

Name Description Syntax Example
Include tag Only show posts matching the specified tag tag_name example
Exclude tag Only show posts without the specified tag -tag_name -example
Special filter Match a metadata piece directly ~field:[op]value ~size:>1000

Special filters

Special filter match a piece of metadata against a value or set a parameter. The following filters are supported:

Filter Description Supported ops Possible values Example
sort Field to sort results by None creation_date, filesize, width, height ~sort:width
rating Rating category None safe, questionable, explicit, violent ~rating:safe
size File size in bytes =, <, > any positive integer ~size:>1048576
mime MIME types None any supported MIME type ~mime:image/png
width Width in pixels =, <, > any positive integer ~width:<1280
height Height in pixels =, <, > any positive integer ~width:<720

Examples

1920x1080 wallpaper - safe for work, easy on the eyes, and not a screenshot:

~rating:safe ~width:=1920 ~height:=1080 wallpaper -bright -screenshot

Images of animals, but not birds:

animal -avian -bird -human

Close-up images of blue-eyed smiling people:

face smile blue_eyes -animal
